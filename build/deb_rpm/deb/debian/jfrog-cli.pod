=head1 NAME

jfrog-cli - Smart client that provides a simple interface that automates access to JFrog products.

=head1 SYNOPSIS

jfrog-cli 

=head1 DESCRIPTION
 JFrog CLI is a compact and smart client that provides a simple interface that automates access to JFrog products 
 simplifying your automation scripts and making them more readable and easier to maintain.
